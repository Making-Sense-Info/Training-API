# Training API

Java API for training purpose.

## Run locally

Execute `mvn spring-boot:run` to expose API on `http://localhost:8080`.

Swagger interface will also be available on `http://localhost:8080/swagger-ui`.

## Docker

## Endpoints

- GET `/people`
- GET `/people/id`
- POST `/people`
- DELETE `/people/id`
- POST `/people/reset`
